Mes Stages

Voici les missions que j’ai effectuées durant mes deux stages de BTS SIO SLAM.

Stage 1 — Développement Web

Entreprise : Leano Design

Période : Mai - Juin 2025

Mission principale : Développement d’un site web de type vitrine pour présenter les services de l'agence et optimiser la visibilité en ligne.

Objectifs :

  • Créer une interface responsive et accessible en HTML / CSS / JS
  • Optimiser le référencement naturel (SEO) et les performances
  • Mettre en place une documentation technique pour la maintenance
  • Collaborer avec l'équipe design pour respecter la charte graphique

Compétences mobilisées :

  • Développement Front-End
  • Intégration Web (CSS3/Flexbox)
  • Travail en méthode Agile
  • Gestion de versions (Git/GitHub)

Stage 2 — Développement Web

Entreprise : Factor-I

Période : Janvier - Février 2026

Mission principale : Développement et maintenance, avec un focus sur l'automatisation des processus internes.

Objectifs :

  • Concevoir et développer des fonctionnalités back-end
  • Manipuler des bases de données relationnelles
  • Assurer la sécurité des données utilisateurs
  • Participer aux réunions techniques et aux revues de code

Compétences mobilisées :

  • Développement Back-End
  • Bases de données (SQL)
  • Travail en équipe
  • Analyse des besoins
